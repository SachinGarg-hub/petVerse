# 🐾 PetVerse — A Social Universe for Pet Lovers

PetVerse is a modern full-stack social media platform built for pet lovers and pet owners. It provides a space where users can share their pets’ moments, connect with a community, and explore pet adoption—all in one place.

---

## 🚀 Features

* 🐶 **Pet-Centric Feed** – Share posts, photos, and reels
* 📸 **Media Uploads** – Upload images and short videos
* ❤️ **Engagement** – Like, comment, and interact with posts
* 💬 **Real-Time Chat** – Connect with other users instantly
* 🏡 **Adoption Module** – Help pets find loving homes
* 🔐 **Authentication** – Secure login and user management
* 🎨 **Aesthetic UI** – Clean and responsive design

## ⚙️ Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/your-username/petverse.git
cd petverse
```

2. **Install dependencies**

```bash
cd client
npm install

cd ../server
npm install
```

3. **Setup environment variables**
   Create a `.env` file in the server folder and add:

```
MONGO_URI=your_mongodb_uri
PORT=5000
JWT_SECRET=your_secret_key
```

4. **Run the project**

```bash
# Start backend
cd server
npm run dev

# Start frontend
cd client
npm start
```
